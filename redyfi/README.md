# redyfi - dy.fi IP refresher

Simple app to update [dy.fi](https://www.dy.fi) DynDNS IP address for a hostname.

### Install

Clone this repository and install by running ```go install```.

### Usage

```
[path]:
  -hostname string
        hostname to update
  -mail string
        email address for user agent header
  -password string
        dy.fi password
  -username string
        dy.fi username
```
